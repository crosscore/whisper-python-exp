## プロジェクト概要
- プロジェクト名：Whisper音声認識アプリ
- 目的：Whisperを使用した音声認識アプリの開発
- 開発環境：
  - OS: MacOS
  - IDE: VSCode
  - Python環境: venv

## 実行方法
1. 環境のセットアップ
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. アプリケーションの起動
```bash
cd src
streamlit run app.py
```

## 注意点
- ffmpegのインストールが必要です
- モデルの初回ダウンロードには時間がかかる可能性があります
